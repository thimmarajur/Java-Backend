# âœ… **What Is an API Gateway?**

**API Gateway is a single entry point** for all client requests in a microservices architecture.
Instead of a client calling multiple microservices directly, **all requests first go through the Gateway**, which then routes them to the appropriate service.

Think of it like a **front door** to all your backend services.

---

Below are the real reasons interviewers expect:

## **1. Single Entry Point for All Clients**
